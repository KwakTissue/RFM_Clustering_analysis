# 온라인 쇼핑몰 매출 증가 원인 파악 프로젝트 - 고객 행동패턴 이해
> 프로젝트 진행 기간 : 2024.06.20~2024.06.27 \

0. **문제정의**

* Key ask : 현재의 좋은 매출를 유지하기 위한 고객의 행동패턴 이해

* 고객 세분화
* 고객 구매 주기 분석
